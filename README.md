## 📖 简介

这是一款综合性的 Windows 硬件检测与系统维护工具，集成了硬件诊断、驱动管理、系统修复、安全监控等多种功能。界面简洁直观，操作便捷，适合普通用户和技术人员使用。

### 核心优势

- ✅ **全面硬件检测** - 支持所有 Windows 版本，包括 Windows XP
- ✅ **智能驱动管理** - 自动检测和安装缺失驱动
- ✅ **系统修复工具** - 集成 DISM、VC++ 运行库修复
- ✅ **实时监控** - 内存优化、崩溃监控、BSOD 检测
- ✅ **安全防护** - 网络防火墙、USB 安全检测、进程保护
- ✅ **低资源占用** - 优化设计，RAM 占用 < 300MB

## 📦 安装与运行

### 方法一：直接运行（推荐）

1. 下载程序文件
2. 解压到任意目录
3. 双击 `HardwareDiagnostics.exe`
4. 以管理员身份运行以获得完整功能

### 方法二：编译源码

```bash

# 进入项目目录
cd HardwareDiagnostics

# 编译 Release 版本

# 发布程序
```

## 🔧 功能模块详解

### DISM 命令详解

#### 系统映像维护

1. **扫描系统健康**
   ```
   DISM /Online /Cleanup-Image /ScanHealth
   ```
   - 用途：扫描系统映像文件是否有损坏
   - 执行时间：5-20 分钟
   - 风险等级：低

2. **修复系统映像**
   ```
   DISM /Online /Cleanup-Image /RestoreHealth
   ```
   - 用途：自动检测并修复损坏的系统文件
   - 执行时间：10-60 分钟
   - 风险等级：低

3. **深度清理并重置**
   ```
   DISM /Online /Cleanup-Image /StartComponentCleanup /ResetBase
   ```
   - 用途：深度清理，删除所有旧版本组件
   - 执行时间：10-60 分钟
   - 风险等级：中（清理后无法卸载更新）

#### Windows 功能管理

4. **启用.NET Framework 3.5**
   ```
   DISM /Online /Enable-Feature /FeatureName:NetFx3 /All
   ```
   - 用途：安装.NET Framework 3.5
   - 执行时间：2-10 分钟
   - 风险等级：低

### 安全中心配置

#### 网络防火墙设置
- **启动防火墙**：开始监控网络数据包
- **停止防火墙**：停止监控
- **查看数据包**：实时显示可疑数据包
- **威胁等级**：低/中/高/严重

#### USB 安全设置
- **启动 USB 监控**：开始监控 USB 设备插入
- **停止 USB 监控**：停止监控
- **查看设备列表**：显示所有插入的 USB 设备
- **风险等级**：低/中/高/严重

#### 进程保护设置
- **启用进程保护**：启动多层防护
- **禁用进程保护**：停止防护
- **查看保护状态**：显示各项保护状态

### 下载工具配置

#### 支持的下载工具
- **内置下载器**：.NET Framework 自带，无需安装
- **curl**：需要单独安装，支持断点续传
- **wget**：需要单独安装，支持递归下载
- **aria2**：需要单独安装，支持多线程
- **自定义工具**：手动指定路径

#### 线程数设置
- 范围：1-256
- 推荐值：8-16
- 大文件：可使用更高线程数

## ❓ 常见问题

### Q1: 程序无法启动？
**A**: 请确保已安装 .NET Framework 4.8 或更高版本。下载地址：https://dotnet.microsoft.com/download/dotnet-framework

### Q2: 某些功能无法使用？
**A**: 请以管理员身份运行程序。右键点击程序图标，选择"以管理员身份运行"。

### Q3: DISM 命令执行失败？
**A**: DISM 需要 Windows 7 或更高版本，并且需要管理员权限。

### Q4: 下载管理器提示 curl/wget 未找到？
**A**: 这是正常提示。您可以：
- 安装 curl 或 wget
- 使用内置的 .NET 下载器
- 手动指定其他下载工具路径

### Q5: RAM 清理后内存没有减少？
**A**: 某些系统进程无法清理，这是正常现象。清理主要针对应用程序进程。

### Q6: 安全中心误报？
**A**: 某些正常软件可能触发安全检测。可以将可信软件添加到白名单。

### Q7: 支持 Windows XP 吗？
**A**: 是的，核心功能支持 Windows XP。但部分功能（如 DISM）需要 Windows 7 或更高版本。

### Q8: 如何更新程序？
**A**: 下载最新版本，覆盖旧版本即可。配置文件和日志会保留。

## 🛠️ 技术栈

- **开发语言**: C# 10.0
- **框架**: .NET Framework 4.8
- **UI 框架**: Windows Forms
- **兼容性**: Windows XP - Windows 11
- **架构**: 模块化设计，6 个子项目

### 项目结构

```
HardwareDiagnostics/
├── src/
│   ├── HardwareDiagnostics/          # 主程序入口
│   ├── HardwareDiagnostics.Core/     # 核心功能库
│   ├── HardwareDiagnostics.Hardware/ # 硬件检测模块
│   ├── HardwareDiagnostics.System/   # 系统维护模块
│   ├── HardwareDiagnostics.Monitoring/ # 监控诊断模块
│   ├── HardwareDiagnostics.Security/ # 安全中心模块
│   └── HardwareDiagnostics.UI/       # 用户界面模块
├── publish/                          # 发布输出目录
└── README.md                         # 项目说明文档
```

### 核心技术

- **WMI (Windows Management Instrumentation)**: 硬件信息获取
- **SetupAPI**: USB 设备管理
- **Raw Socket**: 网络数据包捕获
- **Windows NT API**: 进程保护
- **Performance Counter**: 系统性能监控
